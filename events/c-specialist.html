C’pecailist

Description

/* C'pecialist is the ultimate battle to demonstrate one's programming skills ; you and your teammate will face fun and exciting challenges that will put your logic, debugging, and C  programming skills to the test. ;  So what are you waiting for? Do you think you've got what it takes? */

Round 1 : Blind Coding & Debugging (qualifying round) Blind coding : 1st team member will be provided with a problem statement which has to be solved using c with the screen turned off within 30 mins. Debugging : 2nd team member will have to get the output of a given program and debug a c-file. The output of the given program will be the password of the c-file.

Round 2 : C-battle Participants will have to solve the given set of problem statements. (c language)

Judging Criteria

Blind coding :  They will be judged based on the accuracy of the code. At the start of blind coding round, all the participants will be given 50 points and for every error 1 point will be

Debugging :  10 points will be awarded to the team who gets output of the code If a team fails to get the program output after 10 mins they can request the coordinator for password but 5 points will be deducted from their score. For each bug a team fixes will be awarded with 2 points.  For errors (bugs created)  by participants 2 points will be deducted

C-battle The winning teams will be decided based on the time they take to solve the given set of problems. In case a tie arises between teams a tie breaker round will be conducted to select the winner.

Rules & Regulations

No internet access will be provided

Participants must carry their own pen

The Judges decision will be considered final
